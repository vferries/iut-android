AsyncTask

  • C’est a qu’on trouve dans la documentation en premier
  • Combinaison d’une action longue et d’un bout de code excuter dans le thread UI 
AsyncTask<String, Integer, Bitmap> task =
    new AsyncTask<String, Integer, Bitmap>() {
  @Override
  protected void onPreExecute() {
    /* initialisation */
  }
  @Override
  protected Bitmap doInBackground(String... params) {
      /* Traitement long */
      return null;
  }
  @Override
  protected void onPostExecute(Bitmap result) {
    /* UI Thread */
  }
  @Override
  protected void onProgressUpdate(Integer progress) {
    /* Update progress */
  }
};
task.execute();
  • Petits soucis d’ordonnancement, donc pas prconis par les experts
