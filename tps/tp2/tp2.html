TP2

Mise en place d'une interface graphique.

Objectif de ce TP

  • Crer votre premire interface simple en partant de zro
  • Grer les layouts, le positionnement…
  • Ajouter du comportement

Projet

  • Crez un nouveau projet
    • API lvl 15 minimum
    • API lvl 28 en target
  • Utilisez encore une Empty Activity

Petite astuce

  • Pour le sparateur, mettre une View toute simple avec :
    • la mme largeur que son parent
    • une hauteur de 1dp
    • un background de couleur #656565
  • Pour l’image, pour l’instant on la rajoute au rpertoire de ressources
  • Vous pouvez le faire en mode textuel en basculant de visualisation en bas de la fentre

Ajout de comportement

  • Maintenant qu’on a une interface un peu plus complte, on va rajouter une action notre bouton.
  • Pour ce faire, on va appeler le code suivant dans l'vnement clic qu’on va attacher au bouton : 
Intent intent = new Intent(Intent.ACTION_VIEW);
intent.setData(Uri.parse("https://github.com/vferries/iut-android"));
startActivity(intent);

Rappel raccourcis utiles sous Android Studio

  • Ctrl + N pour afficher une classe partir de son nom
  • Ctrl + H pour voir toutes les classes qui implmentent ou tendent (type hierarchy)
  • Alt + Entre pour les imports et correctifs divers
